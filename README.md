This project focuses on analyzing transactional sales data for a coffee shop to generate actionable insights that support business decision-making. The objective was to identify top-performing products, understand sales trends across different time intervals, and provide recommendations to improve revenue. I began by designing a data flow diagram in Miro to outline the data pipeline from source to reporting. The raw data was then processed in Databricks using SQL, where I cleaned and transformed the dataset, handled inconsistencies, created calculated fields such as total_amount (unit_price × transaction_qty), and grouped data by product type and time intervals. The processed data was exported to Excel, where I built pivot tables, charts, and dashboards to visualize key trends. The analysis revealed high-performing products, peak sales periods, and underperforming items, leading to recommendations such as increasing stock for best-selling products, running promotions during low-demand periods, and improving product visibility. Tools used in this project include Miro for planning, Databricks for data processing, Excel for analysis and visualization, and PowerPoint for presenting insights.# Bright-Coffer-Shop-analysis
